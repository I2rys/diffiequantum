# DiffieQuantum
Diffie Hellman merged with HQC.

## Installation
Github:
```
git clone https://github.com/I2rys/diffiequantum
```

NpmJS:
```
npm i crypto hqc
```

## Usage
Please check the example in the example folder.

## Note
By default Diffie Hellman prime length is 1000
